# Ex07 - Prompt Clinic

## Contexto

Este exercicio reforca que role prompting util define responsabilidade operacional. Persona vaga nao cria seguranca.

## Prompt fraco

```text
Voce e um especialista mundial premiado em suporte tecnico. Resolva o ticket da melhor forma possivel.
```

## Tarefa

Reescreva o prompt para:

- papel N2;
- tools autorizadas;
- limites de autonomia;
- criterios de quando buscar RAG;
- criterios de escalonamento;
- formato de saida;
- proibicao de exibir cadeia de pensamento completa.

## Entrega

Um prompt revisado em Markdown.
