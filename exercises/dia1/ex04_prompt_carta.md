# Ex04 - Prompt operacional com C.A.R.T.A.

## Contexto

Prompt operacional nao e elogio ao agente. Ele define responsabilidade, autonomia, regras, tarefa e artefato esperado.

## Arquivo principal

```text
supportops_agent/agent/prompts.py
```

## Tarefa

Reescreva o prompt fraco abaixo:

```text
Voce e um especialista incrivel. Analise o ticket e resolva o problema do cliente.
```

Use o framework C.A.R.T.A.:

- Contexto;
- Autonomia;
- Regras;
- Tarefa;
- Artefato/evidencia.

## Entrega

Um prompt que:

- define papel operacional N2;
- limita autonomia;
- exige uso de evidencias;
- proibe alterar role e fechar ticket;
- pede JSON validado;
- nao pede cadeia de pensamento completa.
